# Convert dsem to sem output

Convert output from package dsem to sem

## Usage

``` r
as_sem(object, lag = 0)
```

## Arguments

- object:

  Output from
  [`dsem`](https://james-thorson-NOAA.github.io/dsem/reference/dsem.md)

- lag:

  what lag to extract and visualize

## Value

Convert output to format supplied by
[`sem`](https://rdrr.io/pkg/sem/man/sem.html)
