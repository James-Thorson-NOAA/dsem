# Isle Royale wolf and moose

Data used to demonstrate and test cross-lagged (vector autoregressive)
models

## Usage

``` r
data(isle_royale)
```

## Details

Data extracted from file "Data_wolves_moose_Isle_Royale_June2019.csv"
available at <https://www.isleroyalewolf.org> and obtained 2023-06-23.
Reproduced with permission from John Vucetich, and generated by the
Wolves and Moose of Isle Royale project.

## References

Vucetich, JA and Peterson RO. 2012. The population biology of Isle
Royale wolves and moose: an overview. <https://www.isleroyalewolf.org>
