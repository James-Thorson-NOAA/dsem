# Print fitted dsem object

Prints output from fitted dsem model

## Usage

``` r
# S3 method for class 'dsem'
print(x, ...)
```

## Arguments

- x:

  Output from
  [`dsem`](https://james-thorson-NOAA.github.io/dsem/dev/reference/dsem.md)

- ...:

  Not used

## Value

No return value, called to provide clean terminal output when calling
fitted object in terminal.
